# Architecture Decision Records

Records of significant decisions: the context, the options weighed, the choice,
and the consequences. Code shows *what*; an ADR preserves *why*. Managed by the
`/adr` skill. Each ADR is immutable once `accepted` — change a decision by
superseding it, never by rewriting history.
